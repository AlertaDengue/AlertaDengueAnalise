Profissionalizando os testes
devtools::use_testthat()
This will: create a tests/testthat directory, adds testthat to the Suggests field in the DESCRIPTION,
creates a file tests/testthat.R that runs all your tests are when R CMD check runs. 

Agora:
Modify your code or tests.
Test your package with Ctrl/Cmd + Shift + T or devtools::test().
Repeat until all tests pass.
